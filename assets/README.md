# Assets

This directory contains static assets for the Clutterless Chrome extension.

## Structure

- `screenshots/` - Application screenshots for documentation
  - `clutterless-screenshot.png` - Main interface screenshot used in README

## Usage

These assets are referenced in documentation and should be kept organized by type. When adding new assets:

1. Create appropriate subdirectories by asset type
2. Use descriptive, kebab-case filenames
3. Update this README to document new assets
